# Global Superstore Data Cleaning (Excel)

## Project Overview
This project focuses on cleaning and standardizing a Global Superstore sales dataset using Microsoft Excel.

## Issues Identified
- Inconsistent date formats
- Mixed discount values (percent and decimal)
- Currency symbols in sales column
- Negative profit values
- Missing and blank values

## Cleaning Steps
- Standardized date format (YYYY-MM-DD)
- Converted discount values to decimal format
- Removed currency symbols and fixed numeric columns
- Handled negative profits and blanks
- Created separate raw and cleaned datasets

## Tools Used
- Microsoft Excel

## Files
- `data/raw/Global_Superstore_Raw_Data.xlsx`
- `data/cleaned/Global_Superstore_Cleaned_Data.xlsx`
